# MbokaStream 🎬

> **Biso na biso — La diaspora congolaise mérite mieux.**

A Netflix-style streaming platform for Congolese TV shows, built for the diaspora in USA, Belgium, France, and Canada.

## Color Palette

Inspired by the official flags of the **DRC** and **Republic of Congo (Brazzaville)**:

| Color | Hex | Meaning |
|-------|-----|---------|
| Sky Blue | `#007FFF` | DRC — Paix / Peace |
| Red | `#EF3340` | Both flags — Sang des martyrs |
| Gold | `#FFD100` | Both flags — Richesse & Noblesse |
| Forest Green | `#009543` | Congo-Brazza — Forêts |

## Content Partner

**SB Production** (Sila Bisalu Officiel) — Kinshasa, DRC  
Series: Sans Pitié, Le Poids du Silence, Destin Brisé, La Demoiselle de la Mort, Témoin Coupable, Mariage du Péché

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: CSS Modules
- **Fonts**: Syne (display) + Outfit (body)
- **Video**: Mux (planned)
- **Payments**: Stripe (planned)
- **Auth**: Clerk (planned)
- **Deploy**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
